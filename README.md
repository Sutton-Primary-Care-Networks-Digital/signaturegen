# Email Signature Generator

A simple, standalone HTML-based email signature generator. Fill in your details and generate a professional email signature that can be copied directly into your email client.

## Features

- **Easy Form Input**: Enter your name, title, company, and contact information
- **Professional Layout**: Generates a clean, corporate-style email signature
- **Customizable**: Includes fields for phone, email, website, and address
- **Copy to Clipboard**: One-click copying of the generated HTML signature
- **Responsive Design**: Works on different screen sizes
- **No Dependencies**: Single HTML file with embedded CSS and JavaScript

## Usage

1. Open `index.html` in any web browser
2. Fill in your personal and professional information
3. Click "Generate Signature" to preview
4. Click "Copy to Clipboard" to copy the HTML
5. Paste the HTML into your email client's signature settings

## Customization

The signature template can be customized by editing the HTML and CSS in `index.html`. The styling uses the Aptos font family with fallbacks to Arial and sans-serif.

## Browser Compatibility

Works in all modern web browsers including:
- Chrome
- Firefox
- Safari
- Edge

## Deployment

Since it's a single HTML file, you can:
- Host it on any web server
- Share the file directly
- Embed it in other web applications
